KMP 的算法

KMP 算法，基本思想是，當出現不匹配時，就能知曉一部分文本的内容（因爲在匹配失敗之前它們已經和模式相匹配）。我們可以利用這些信息避免將指針回退到所有這些已知的字符之前。這樣當出現不匹配時，可以提前判斷如何重新開始查找，而這種判斷只取決於模式本身。

直接插入排序

直接插入排序的基本思想是： 把數組 a[n] 中待排序的 n 個元素看成為一個有序表和一個無序表，開始時有序表中只包含一個元素 a[0]， 無序表中包含 n - 1 個元素 a [1…(n - 1)] ，排序過程中每次從無序中取出第一個元素，把它插入到有序表中的適當的位置，使之成爲新的有序表，這樣經過 n - 1 次插入后，無序表就變爲空表，有序表中就包含了全部 n 個元素，至此排序完畢。在有序表中尋找插入位置是采用從后向前的順序查找的方法。

希爾插入排序

從對直接插入排序的分析得知，其算法時間複雜度為 $O(n^{2})$，但是，若待排序記錄序列為“正序”時，其時間複雜度可提高至 $O(n)$。由此可設想，若待排序記錄序列按關鍵字“基本有序”，即序列中具有下列特性

$$
R_{i}.key < max {R_{j}.key}, j < j
$$

的記錄較少時，直接插入排序的效率就可大大提高，從另一方面來看，由於直接插入排序算法簡短，則在 n 值很小時效率也比較高。希爾排序正式從這兩點分析出發對直接插入排序進行改進得到的一種插入排序方法。

希爾排序的基本思想是： 設待排序元素序列有 n 個元素，首先取一個整數 $ gap = \lfloor \frac{n}{3} \rfloor + 1 $ 作爲間隔， 將全部蒜素分爲 gap 個子序列，所有距離爲 gap 的元素放在同一個子序列中，在每一個子序列中分別進行直接插入排序。然後縮小間隔 gap， 取 $ gap = \rfloor \frac{gap}{3} \lfloor + 1 $，重複上述的子序列劃分和排序工作，直到最後取 $ gap = 1 $，將所有元素放在同一個序列中排序爲止。

快速排序

快速排序的基本思想是任取待排序元素序列中的某個元素作爲基準（通常會選擇第一個元素），按照該元素的關鍵字大小，將整個元素序列劃分為左右兩個子序列：左側子序列中所有元素的關鍵字都小於基準元素的關鍵字，右側序列中所有元素的關鍵字都大於或等於基準元素的關鍵字，基準元素則排在兩個子序列中間（這也是該元素最終應該安放的位置）。然後分別對這兩個子序列重複實施上述算法，直到所有的元素都排在相應位置爲止。
